# Command nettoyer

## Details

<!-- --8<-- [start:details] -->
Description: Remove all unpinned messages in the current text channel.

Required permission: `Administrator`
<!-- --8<-- [end:details] -->

## Usage

<!-- --8<-- [start:usage] -->
Remove all unpinned messages in the current text channel:

```text
/nettoyer
```
