# Rôle Voyante

|                              |                                                                                                                                                                |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Identifiant de configuration | `roleSeerId`                                                                                                                                                   |
| Code                         | `voy`                                                                                                                                                          |
| Type                         | Villageois                                                                                                                                                     |
| Description                  | Chaque nuit, la voyante peut observer le rôle d'un autre joueur. Les rôles `Amoureux`, `Envoûté`, `Imbibé d'essence` et `Infecté` ne lui sont pas communiqués. |
